# Module statistiques

Feature admin qui **reproduit les indicateurs du rapport d'activité** de l'association. Objectif : éliminer la compilation manuelle que le client faisait dans des fichiers Excel.

Filtre par année systématique — toutes les métriques se recalculent selon l'année sélectionnée.

## Indicateurs — Profils & situations

| Indicateur | Description |
|---|---|
| Jeunes inscrits | Nombre de jeunes inscrits à ≥ 1 action (+ évolution vs N-1) |
| Âge médian | Des jeunes inscrits |
| Répartition par sexe | % femmes / % hommes |
| Répartition par situation | sans emploi / scolarisé ordinaire / médico-social / emploi-formation / autre |

## Indicateurs — Provenances

| Indicateur | Description |
|---|---|
| Origine géographique | % de jeunes de la métropole de Lyon (via `postal_code`) |
| Structures prescriptrices | Nombre total d'organismes, dont structures Prado |
| Accompagnement | Camembert ASE / PJJ / Handicap / décrochage / RSJ / CEJ / Autre |

## Indicateurs — Actions

| Indicateur | Description |
|---|---|
| Nombre d'actions | Total + répartition par catégorie/programme (anneau) |
| Taux de participation | % présents / inscrits (via [[concepts/emargement]]) |
| Jeunes par action | Moyenne |
| Fidélisation | % jeunes à 1, 2, 3+ actions |

Lot 5 — **2 j / 1 200 €**. Voir [[sources/devis-v2]].
